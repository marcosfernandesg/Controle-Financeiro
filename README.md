# Sistema de Controle Financeiro em Java

Projeto criado para praticar Java, lógica de programação e orientação a objetos no primeiro semestre de ADS.

## Funcionalidades
- Adicionar transações
- Listar transações
- Remover transações
- Exibir saldo total

## Tecnologias utilizadas
- Java 17
- IntelliJ IDEA
- Programação Orientada a Objetos (POO)

## Como executar
```bash
javac Main.java
java Main
